# numpy-tiff-deepimagej
Test images (converted to tiff from numpy) for testing bioimage.io models in DeepImageJ 

# Contents of the folders
Every model (resource_id, version) has a folder with 4 files:
- `input_shape.edn`
- `output_shape.edn`
- `sample_input_0.tif`
- `sample_output_0.tif`
